=================
Bold, Italic etc.
=================


..  index:: reST; Bold

Bold
====

..  code-block:: rest

    This is normal text. **This is bold text**

How it looks:

This is normal text. **This is bold text**


..  index:: reST; Italic

Italic
======

..  code-block:: rest

    This is normal text. *This is italic text.*

How it looks:

This is normal text. *This is italic text.*


Possible Errors
===============

..  important::

    Make sure there is no space between the markup and the styled text,
