==========================
Problems With Inline Style
==========================


If you use **bold** or *italic*, make sure there is no space between
the markup and the styled text.


Correct Syntax
--------------

::

   This is normal text. **This is bold text.**

How it looks:


This is normal text. **This is bold text.**

Wrong Syntax
------------

::

   This is normal text. ** This is bold text.**

How it looks:


This is normal text. ** This is bold text.**
